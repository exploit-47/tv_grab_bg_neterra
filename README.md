tv_grab_bg_neterra
==================

XMLTV epg grabber for bulgarian channels streamed by neterra.tv

Requirements
- java > 1.7

Usage arguments:

Option        Description
------        -----------
-d [Integer]  number of EPG days to get (default: 5)
-h            show help
-o <File>     output filename
-t            Timezone. Example GMT+1 or GMT-5
                (default: GMT)
